Claro Lens — Settings
Claro Proxy Configuration
Your proxy worker URL (e.g. https://claro-proxy.yourname.workers.dev)
Your Cloud Run PDF extractor URL — optional, enables automatic PDF analysis
The value of CLARO_APP_SECRET — shared by the Worker and PDF extractor

The extension talks directly to your own Cloudflare Worker proxy — the same one the iOS app uses. Your Anthropic API key stays in the Worker; only the app secret is stored locally in browser storage. To find your secret, run wrangler secret list in the cloudflare-worker/ directory.
